Deskripsi Project
Sistem Informasi Jadwal Perkuliahan Gedung A10 Teknik Informatika Universitas Negeri Surabaya adalah sebuah platform digital terintegrasi yang dirancang untuk mengelola, menampilkan, dan memantau jadwal perkuliahan di Gedung A10 secara real-time dan terpusat. Sistem ini bertujuan untuk menyediakan akses informasi jadwal yang akurat, mudah diakses, dan dapat diperbarui secara dinamis oleh admin, dosen, maupun mahasiswa.
Melalui sistem ini, pengguna dapat:
•	Melihat jadwal perkuliahan per hari, per ruangan, atau per mata kuliah.
•	Mengakses informasi dosen pengampu, kelas dan lokasi ruangan.
•	Memantau ketersediaan ruangan secara real-time.

Fitur utama sistem meliputi:
1.	Dashboard Admin: untuk mengelola data jadwal, ruangan, dosen, dan mata kuliah.
2.	Tampilan Jadwal Interaktif : dengan filter berdasarkan hari, ruang, atau kelas.
3.	Pencarian Jadwal: berdasarkan mata kuliah, dosen, atau ruangan.
Sistem ini dibangun dengan teknologi web modern yang responsif, aman, dan user-friendly, sehingga dapat diakses melalui berbagai perangkat. Diharapkan sistem ini dapat meningkatkan efisiensi pengelolaan jadwal akademik serta mendukung proses belajar mengajar yang lebih teratur dan transparan.
